h3. requirements / installation

Compiles against Scala 2.8.0.RC1 and Java 1.6. The repository contains a project file for Intellij IDEA 9 CE, and project files for sbt. Targets for sbt:

* @clean@
* @compile@
* @doc@
* @package@
* @run@
* @standalone@

The dependancies currently need to be installed manually. Your @lib@ folder should contain

* @ScissLib.jar@ (from "sourceforge.net/projects/scisslib":http://sourceforge.net/projects/scisslib/ )
* @ScissDSP.jar@ (from "github.com/Sciss/ScissDSP":http://github.com/Sciss/ScissDSP )

Furthermore, you need "github.com/Sciss/ScalaInterpreterPane":http://github.com/Sciss/ScalaInterpreterPane and "github.com/Sciss/SonogramOverview":http://github.com/Sciss/SonogramOverview . You can make both available, by cloning their gits and running @sbt publish-local@ in their respective folders.

To create and run a self-contained jar:

pre. $ sbt standalone
$ java -jar TemporalObjects-full.jar
