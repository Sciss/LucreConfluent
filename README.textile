h3. requirements / installation

Compiles against Scala 2.8.0 and Java 1.6. The repository contains a project file for Intellij IDEA 9 CE, and project files for sbt. Targets for sbt:

* @clean@
* @compile@
* @doc@
* @package@
* @console@

Note: see tag v0.14 for the example using actual audio file regions and sonogram view. The current version is stripped down to prepare for use as library.
